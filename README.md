# [騎行數據整合與保養監控系統]

## Proposal Report

### 動機與目標
<!-- 說明為什麼想做這個專題 -->
雖然單車的耗材更換與否大多是根據外觀來決定(比如直接測量鍊條拉伸程度、外胎磨損程度)，但不是每個人上路前都會仔細檢查車輛，所以我想透過實際的騎乘數據來了解一項耗材的使用距離，提供直觀的對比(比如A外胎6000公里就不行了，B可以撐10000公里)。

### 競品比較
<!-- 比較目前已經存在可取得的類似工具或應用 -->
有許多騎行數據紀錄工具如strava、velodash...等，但兩者都缺乏針對耗材的里程累積，也缺乏針對旅程的文字、圖片記錄區塊。
### 預期功能
<!-- 列出預計實作的功能 -->
實作出針對耗材的里程累積(鍊條、鍊條油、外胎、把帶、煞車皮、線材、保養)，以及視覺化從strava抓來的數據，提供類似日記的紀錄方式。
